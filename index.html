Vue Server Side Rendering

Vue Server Side Rendering or SSR for short renders all the code in the server rather than on the client, decreasing the load times for your website. When the user clicks on your website instead of having to download all of the HTML, CSS and JavaScript the HTML and CSS will already be saved on the server, so the user does not have a blank screen while waiting for the it all to load.
